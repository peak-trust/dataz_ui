# Audio Assets

This directory contains audio files for the cinematic story experience.

## Required Files

### Ambient Tracks
- `ambient/space.mp3` - Space ambient for Chapter 1
- `ambient/city.mp3` - City ambience for Chapters 2-3
- `ambient/water.mp3` - Waterfront ambience for Chapter 4
- `ambient/orchestral.mp3` - Uplifting orchestral for Chapter 5

### Transition SFX
- `transitions/whoosh.mp3` - Camera zoom transitions
- `transitions/impact.mp3` - Final CTA impact sound

### UI SFX
- `ui/ping.mp3` - Data reveal sounds
- `ui/slide.mp3` - Card slide animations

## Sources

These files should be sourced from royalty-free audio libraries:
- Pixabay (https://pixabay.com/music/)
- Freesound (https://freesound.org/)
- Zapsplat (https://www.zapsplat.com/)

## Notes

- Audio files are optional - the app will gracefully degrade if files are missing
- Recommended format: MP3, 128kbps, mono or stereo
- Keep file sizes under 500KB each for optimal loading
